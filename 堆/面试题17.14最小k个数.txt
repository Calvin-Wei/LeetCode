设计一个算法，找出数组中最小的k个数。以任意顺序返回这k个数均可。

示例：

输入： arr = [1,3,5,7,2,4,6,8], k = 4
输出： [1,2,3,4]
提示：

0 <= len(arr) <= 100000
0 <= k <= min(100000, len(arr))

1.执行时间超过73%
class Solution {
    public int[] smallestK(int[] arr, int k) {
        int array[]=new int[k];
        Arrays.sort(arr);
        for(int i=0;i<k;i++){
            array[i]=arr[i];
        }
        return array;
    }
}
